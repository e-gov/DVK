********************************************************************************
                             Täielik andmebaasiskript
********************************************************************************

NB! Enne andmebaasi skriptide kasutamist loe läbi kliendi paigaldusjuhend:
https://svn.eesti.ee/projektid/dvk/doc/client/dvk_klient_paigaldusjuhend.odt

Täielik ja kõiki viimaseid uuendusi sisaldav andmebaasiskript asub failis
client_latest_[andmebaasinimi].sql

Toetatud andmebaasitarkvarad on
- Microsoft SQL Server 2000 või uuem
- Oracle 10g või uuem
- PostgreSQL 8.0 või uuem
- SQL Anywhere 9.0 või uuem
